git log中文乱码问题

发表于 2020-11-17 | 分类于 Git的学习

参考链接：《git- win10 cmd git log 中文乱码 解决方法》

问题描述：在终端：$ git log之后，commit的中文信息发生乱码

解决步骤如下：

一、 添加环境变量
  1. 任意打开一个文件夹，找到此电脑，右键选择属性，进入计算机属性界面
  2. 选择高级系统设置
  3. 选择高级，环境变量
  4. 用户变量下，选择新建
  5. 输入变量名：LESSCHARSET，值：utf-8，点击确定
  6. 新建完成后，在环境变量界面点击确定后生效
